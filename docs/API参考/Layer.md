## 属性
* ### .id : String Readonly
    楼层ID。
* ### .object3D : GameObject Readonly
    楼层根节点对象。
* ### .viewState : ViewState Readonly
    楼层预设默认观察视角参数。
* ### .sites : Generator Readonly
    楼层POI列表生成器。
* ### .sites.length : Int Readonly
    楼层POI数量。
