## 属性
* ### .id : String Readonly
    POI对象ID。
* ### .scene : Scene Readonly
    POI所属场景。
* ### .layer : Layer Readonly
    POI所属楼层。
* ### .type : Int Readonly
    POI类型。
* ### .number : Int Readonly
    POI号码。
* ### .position : Vector3 Readonly
    POI坐标。
